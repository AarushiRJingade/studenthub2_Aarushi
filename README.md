StudentHub
This project is the Week 2 submission for the ISTE Summer School 2026, following the instructions in the ISTE_WebDev_Week2_Submission_Guidelines.pdf.

Features
1.Interactive Components: Includes a Task Manager and Sticky Notes.  
2.Data Persistence: Uses localStorage so your data is saved even if you refresh the page.  
3.Mobile-Friendly: Built with Tailwind CSS to ensure the layout is responsive on all devices.
